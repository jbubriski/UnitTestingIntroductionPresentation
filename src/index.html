Unit Testing

John Bubriski
What is unit testing?
Running isolated pieces of code and expecting an outcome.
Why do we unit test?
Make sure your code actually works!
Unit testing is your insurance policy.
Let QA focus on important things!
What do we unit test?
As much as we easily can.
How do we unit test?
  • Eliminate dependencies.
  • Write the test.
  • Make an assertion.
  • Run the test.
How do we run unit tests?
  • NCrunch
  • .NET Demon
  • xUnit test runner
  • Resharper (w/plugin)
  • Test Driven .NET
Examples!
  • Write many unit tests.
  • Make assertions.
  • Run your tests often.
